## Exploración y Visualización de Datos con Python

Este repositorio contiene la aplicación desarrollada como parte del Trabajo Final del módulo Exploración y Visualización de Datos con Python del Diploma Business Analyst.

La aplicación fue construida con Streamlit y permite cargar distintos datasets para realizar tareas de exploración, procesamiento y análisis visual de manera interactiva. Además, incorpora gráficos univariados, bivariados, multivariados y temporales, facilitando la identificación de patrones y la comprensión de la información disponible.

### Herramientas utilizadas
- Python
- Streamlit
- Pandas
- Plotly
- Matplotlib
- Seaborn
- GitHub

### Uso de Inteligencia Artificial
Durante el desarrollo del proyecto se utilizó inteligencia artificial como herramienta de apoyo para comprender algunos mensajes de error, resolver dudas sobre la lógica de programación y evaluar alternativas para determinadas visualizaciones. Asimismo, fue útil para complementar la documentación y mejorar algunos aspectos de la implementación. Sin embargo, el diseño, desarrollo, validación y adaptación de la aplicación fueron realizados de manera personal.
